Галлерея Изображений

Проект - Галлерея Изображений будет представлять собой коллаж с картинками. С возможностью сортировки по тегам и дате добавления. Также будет реализована возможность просмотра картинок в виде сетки(grid) или в виде списка(list)

В качестве главной фишки проекта - модальное окно с навигацией. Ещё сайт будет продуман как для десктопов, так и для мобильных устройств
